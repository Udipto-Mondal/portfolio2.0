# 🌟 Professional Portfolio

This is your modern, responsive portfolio website.

## 📁 Project Structure
- `index.html`: Main structure
- `styles.css`: Custom premium styles
- `app.js`: Application logic and rendering
- `data.js`: Centralized data (easy to edit!)
- `assets/`: Your local media files (e.g., your profile photo)

## 🚀 How to Run

### Method 1: Visual Studio Code (Recommended)
1. Open this folder `c:\Users\Udipto\.gemini\antigravity\scratch\portfolio` in VS Code.
2. If you have the **"Live Server"** extension installed, click **"Go Live"** in the bottom right corner of VS Code.
3. Alternatively, right-click `index.html` and select **"Open with Live Server"**.

### Method 2: Command Line (Node.js)
If you have Node.js installed, run:
```bash
npm start
```
This will start a local server at `http://localhost:3000` (or similar).

### Method 3: Direct Open
Just double-click `index.html` to open it in your browser (though some features might need a server later).

## ✏️ How to Edit
To change your info (name, projects, etc.), simply edit:
👉 **`data.js`**

Everything else will update automatically!
